# How to install?

First clone the repository:

```shell

git clone https://github.com/oZaGGo/dvd-java

```

Go to the folder:

```shell

cd dvd-java

```

Then grant execute permission to the script install:

```shell

chmod +x install.sh

```

Then run the script:

```shell

./install.sh

```

Thats all!

Usage: `dvd [--help] [--version]`
